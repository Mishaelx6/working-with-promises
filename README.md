# working-with-promises

This project demonstrates working with JavaScript promises and task execution.

## Files

- `index.js`: Main module with `runTasks` function that executes tasks synchronously.
- `testTask.js`, `testTask2.js`, `testTask4.js`: Example task files.
- `node internals/`: Subdirectory with its own Node.js project (index.js, package.json, worker.js).

## Development Process

1. Created project files including index.js and test tasks.
2. Added .gitignore to ignore node_modules directories.
3. Staged all files with `git add .`.
4. Committed changes with message "Add project files".
5. Pushed to main branch with `git push origin main`.

Note: The `node internals` folder contains a separate git repository and was added as a gitlink.
